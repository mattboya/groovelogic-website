Jsonify Playlist

Extract Spotify playlist metadata directly into a perfectly formatted JSON array.

Open Web App

Description

Jsonify Playlist gives you total control over your music data. Use your Spotify account to instantly access your personal library, or paste the URL of any public playlist. With one click, the tool extracts all the essential track metadata—including song titles, artists, and track length—and packages it into a perfectly formatted JSON array. You can then download the file directly or copy the code to your clipboard for use in your own local files, web apps, or data backups.

Key Features

  • One-Click Metadata Extraction: Instantly extract song titles, artists, and exact second-counts from any Spotify playlist.
  • Personal Library or Public Link: Use your Spotify account for direct access, or simply paste a public playlist URL.
  • Clean JSON Formatting: Generates a simplified, perfectly formatted JSON array of your tracklist, perfect for developers and DJs.
  • Easy Export: Download the file directly or copy-paste the data straight to your clipboard for immediate use.

Tech Stack

  • Frontend: Next.js 15 (App Router), TypeScript, and Tailwind CSS.
  • Backend & Auth: NextAuth.js for seamless Spotify OAuth, backed by Firebase for data storage and tracking.
  • Integrations: Spotify Web API (metadata), Google Drive API (direct cloud exports), and Polar.sh (subscription & payment processing).
